# 📖 About

Unlike repositories that list hundreds of random questions, this project is organized around **interview patterns**.

The goal is to understand **how to think**, not what to memorize.

Every topic includes:

- 📘 Theory
- 🧩 Pattern recognition
- 🐢 Brute-force approach
- ⚡ Optimized solution
- ☕ Java implementation
- ⏱️ Time & Space Complexity
- 💡 Interview tips
- 🔗 Related LeetCode problems

# 🎯 Why Pattern-Based Learning?

Most interview questions are variations of a limited number of patterns.

Learning those patterns allows you to solve many unseen problems with confidence instead of memorizing individual solutions.
